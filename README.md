# 适配于若依的 React 版本前端
## 基础框架：Ant Designer Pro
[Ant Design Pro](https://pro.ant.design/) 是基于 Ant Design 设计体系的 React UI 组件库，主要用于快速开发企业后台产品原型。
## 微服务切换
1、配置 config/proxy.ts 中的代理

2、请根据实际情况修改 src/services/ant-design-pro 中相关接口的地址
## 安装依赖
```bash
# 推荐：
yarn
# npm
npm install
```
## 启动项目
```bash
npm run dev
```
